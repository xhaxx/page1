# 本地启动指南

> 论坛功能（Spring Boot + React），本地开发/运行环境。

---

## 每日启动步骤

### 第一步：启动后端 API

打开第一个终端：

```bash
cd E:\message-board-api
mvn spring-boot:run
```

后端运行在 **http://localhost:2005**

### 第二步：启动前端（新开终端）

打开第二个终端：

```bash
cd E:\page1
npm run dev
```

前端运行在 **http://localhost:5173**

### 第三步：访问

浏览器打开 **http://localhost:5173** → 滚动到 Section 5 → 点击「进入论坛」

---

## 停止服务

两个终端分别按 `Ctrl + C` 即可。

---

## 常见问题

| 问题 | 解决 |
|------|------|
| 后端启动报端口占用 | 检查 2005 端口：`netstat -ano \| findstr 2005`，杀掉占用进程 |
| 前端连不上后端 | 确认后端已启动，检查 `src/api/forum.ts` 中 `API_BASE` 是否为 `http://localhost:2005/api` |
| 数据库被重置 | `schema.sql` 使用 `CREATE TABLE IF NOT EXISTS`，不会覆盖数据 |
| Java 版本不对 | 需要 JDK 17，检查 `java -version` |
